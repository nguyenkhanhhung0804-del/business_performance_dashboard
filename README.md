# 📊 Power BI Business Performance Dashboard

## Overview

This project presents an interactive **Power BI Business Performance Dashboard** that enables users to monitor key performance indicators (KPIs) and analyze business trends through dynamic visualizations. The dashboard is designed to support data-driven decision-making by providing an executive-level overview of financial and operational performance.

## Objectives

* Build an interactive dashboard for business performance analysis.
* Track KPIs across different reporting periods.
* Identify trends, patterns, and areas for improvement.
* Enable stakeholders to explore data through interactive filters and visualizations.

## Dashboard Features

* 📈 KPI Cards for high-level business metrics
* 📊 Waterfall Chart to analyze performance changes
* 🎯 Scatter Plot for relationship analysis
* 🌳 Treemap for category-level comparisons
* 📉 Line & Column Charts to visualize trends over time
* 🎛 Interactive slicers for year and metric selection
* ⚡ Dynamic measures using DAX

## Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Power Query
* Data Modeling
* Interactive Data Visualization
